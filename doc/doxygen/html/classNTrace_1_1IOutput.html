Detailed Description

Interface for NTracer output.

An IOutput interface takes a log message and does something useful with it. It can either be a filter, writing to a file, store it in a database, printing to stdout, et cetera. IOutputs are also responsible for formatting the output (apart from the message, which is preformatted at the moment of creation).
